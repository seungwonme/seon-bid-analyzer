# 입찰 공고 분석기 (선엔지니어링)

3회차 보충 수업에서 함께 만든 결과물입니다. 나라장터 입찰 공고 PDF를 올리면 핵심 정보를 카드로 정리해 주고, 제출서류 체크리스트와 자격서류 작성·내보내기까지 도와주는 웹 도구입니다.

> 지난 시간 코드를 잃어버렸거나 끝까지 못 따라오셨어도 괜찮습니다. 여기서 코드를 받아 모두 같은 출발점에서 4회차를 시작합니다.

## 무엇을 하는 도구인가요

- 공고 PDF 업로드 → 공고명, 발주기관, 예정금액, 용역기간, 마감일, 참가자격, 제출서류를 뽑아 정리
- 마감일까지 며칠 남았는지 D-Day 표시 (7일 이내는 빨간색)
- 제출서류 체크리스트와 준비 진행률(%)
- 유사용역 실적, 기술자 경력을 폼으로 작성해 Word(.docx) 문서로 내보내기

로그인 없이 브라우저에서 바로 쓰고, 입력한 데이터는 내 브라우저(localStorage)에만 저장됩니다.

## 코드 받는 방법 (둘 중 하나)

**방법 1. ZIP 다운로드 (터미널이 익숙하지 않다면)**

1. 이 페이지 위쪽 초록색 `Code` 버튼 클릭
2. `Download ZIP` 클릭
3. 압축을 풀고, 그 폴더를 Claude Desktop이나 에디터에서 엽니다

**방법 2. git clone (터미널)**

```bash
git clone https://github.com/seungwonme/입찰공고분석기.git
cd 입찰공고분석기
```

## 실행하는 방법

Claude Code에게 이렇게 부탁하면 알아서 설치하고 실행해 줍니다.

```
이 프로젝트를 실행해줘. 필요한 패키지부터 설치하고 개발 서버를 띄워줘.
```

직접 하려면 터미널에서:

```bash
pnpm install   # (없으면 npm install)
pnpm dev       # (없으면 npm run dev)
```

실행되면 브라우저에서 http://localhost:3000 으로 접속합니다.
